01

이 서비스를 기획하게 된 이유

여행을 가거나, 처음 자취를 시작하면 뭐든 비싸게 사게 될까 불안한 적이 있었습니다. 관광지에서 밥 한 끼에 만 오천 원을 내고 나서 "이게 맞는 가격이었나?" 하고 후회한 경험, 에어컨 청소를 불렀는데 15만 원을 달라고 해서 적정한 건지 판단이 안 됐던 경험이 있었습니다.

그래서 생각했습니다. "내가 내는 가격이 적정한 건지 바로 알려주는 서비스가 있으면 좋지 않을까?"

찾아보니 네이버 쇼핑이나 다나와 같은 가격비교 사이트는 있지만, 대부분 온라인 상품 위주이고 택시비나 수리비 같은 오프라인 서비스 가격은 비교가 안 됩니다. 그래서 이런 부분까지 커버하는 AI 서비스를 직접 만들어보자고 결심했습니다.

02

FairPrice AI가 뭔가요?

한 줄로 요약하면, 상품이나 서비스 이름과 가격을 입력하면 AI가 "이 가격 괜찮은 건지" 알려주는 서비스입니다.

사용 흐름

상품명 + 가격 입력 → AI가 시장 데이터와 비교 → 4단계로 판정 → 절약 팁 제공

예를 들어 "짜장면 9,000원"을 입력하면, 시장 평균(약 7,000원)과 비교해서 "좀 비싼 편이에요"라고 알려주고, 어느 정도가 적정한지, 어떻게 하면 더 저렴하게 먹을 수 있는지까지 알려줍니다.

판정 기준 (4단계)

단계기준의미
저렴평균 이하확실히 싸게 산 경우
적정평균 ±10%시장 평균과 비슷한 합리적 가격
주의평균 초과 ~ 최고가 85%좀 비싸니 비교해볼 것
바가지최고가 85% 초과시장가 대비 너무 비쌈

04

주요 기능

1. AI 가격 분석 (핵심 AI 기능)

사용자가 품목명과 가격을 입력하면, 내장된 시장 가격 DB와 비교해서 적정성을 분석합니다. 단순히 평균과 비교만 하는 게 아니라, 구매 장소(관광지/재래시장/온라인 등)에 따른 보정도 적용했습니다. 관광지에서는 일반보다 35% 정도 비싼 게 보통이니까, 그걸 감안해서 판정하는 식입니다.

2. 자체 제작 가격 데이터베이스

음식, 교통, 수리, 전자제품, 생활용품, 여행 이렇게 6개 카테고리에 걸쳐 127개 이상의 품목 데이터를 직접 조사해서 넣었습니다. 각 품목마다 최저가, 평균가, 최고가 범위를 잡아두었습니다.

3. 절약 팁 & 대안 추천

분석 결과와 함께 절약하는 방법이나 비슷한 대안도 같이 보여줍니다. 예를 들어 "아메리카노 6,500원"이 비싸다고 나오면, "편의점 커피는 1,500원부터"라는 식으로 대안을 알려줍니다.

4. 차트 시각화

카테고리별 바가지 위험도와 평균 가격대를 Chart.js로 차트로 보여줍니다.

05

데이터 구성

데이터는 실제 시장 가격을 조사해서 직접 만들었습니다.

카테고리품목 수대표 품목
음식/외식23개짜장면, 삼겹살, 치킨, 커피 등
교통11개택시, KTX, 주유, 대리운전 등
수리/서비스11개에어컨 청소, 이사, 미용실 등
전자제품10개아이폰, 노트북, 에어팟 등
생활용품10개생수, 계란, 쌀, 화장지 등
여행9개호텔, 렌터카, 테마파크 등

각 품목에는 최저가/평균가/최고가 외에 절약 팁 2개, 대안 추천 2~3개가 포함되어 있습니다.

06

기술 스택

구분기술선택 이유
화면HTML + CSS + JS서버 없이 파일만 열면 바로 돌아감
AI 엔진JavaScript규칙 기반 + 통계 분석 알고리즘 직접 구현
데이터JSON (JS 내장)별도 DB 서버 불필요
차트Chart.js가볍고 깔끔한 차트 라이브러리
디자인 참고노션, 토스증권깔끔한 UI 벤치마킹
별도 설치나 서버 없이 index.html 파일을 브라우저에서 열면 바로 작동합니다.

AI 분석 로직

  1. 사용자가 입력한 품목명을 DB에서 퍼지 매칭으로 검색
  2. 매칭된 품목의 최저/평균/최고가 범위 가져오기
  3. 구매 장소에 따라 가격 보정 (관광지 +35%, 재래시장 -15% 등)
  4. 입력 가격이 보정 범위에서 어디에 있는지 백분위 계산
  5. 4단계 중 하나로 판정하고 절약 가능 금액 산출
07

시장 분석

국내 가격비교 시장은 약 2조 원 규모이고, 스마트 소비 관련 앱 시장은 매년 15% 정도 성장하고 있다고 합니다. 가격비교에 대한 수요 자체는 이미 충분히 검증된 상태입니다.

기존 서비스와 비교

서비스잘하는 점아쉬운 점
네이버 쇼핑상품 많고 검색 편함온라인 물건만 비교됨
다나와전자제품 비교에 특화음식이나 서비스 비교 안 됨
번개장터중고 시세 확인 가능적정가 판단 기능은 없음
FairPrice AI오프라인 서비스까지 커버, AI 판정아직 데이터가 적음
기존 서비스들이 다루지 않는 오프라인 서비스 가격(택시비, 수리비 등)까지 커버하면서, 단순 비교가 아닌 적정/바가지를 판정하는 것이 차별점입니다.

09

앞으로의 발전 가능성

단기 (3~6개월)

  • 지금은 직접 만든 데이터인데, 실시간 크롤링으로 자동 수집하면 훨씬 정확해질 수 있음
  • 모바일 앱으로 만들면 밖에서 바로 확인 가능
  • 사용자가 직접 가격을 제보하는 기능 추가

중장기 (6개월~1년)

  • 카메라로 가격표나 영수증을 찍으면 OCR로 자동 분석
  • LLM을 연결해서 "제주도 2박 3일 적정 예산은?" 같은 질문에도 대응
  • 해외 여행지 데이터도 추가해서 글로벌 서비스로 확장
