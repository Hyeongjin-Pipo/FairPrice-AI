# AI 스타트업 프로젝트 기획서: FairPrice AI

## 1. 프로젝트 요약 (Executive Summary)
**FairPrice AI**는 포켓몬 카드, 한정판 스니커즈, 단종 레고 등 개인 간 거래(C2C) 시장에서 발생하는 **'가격 정보의 비대칭성'을 해결하기 위해 기획된 AI 기반 가치 평가 플랫폼**입니다.

기존 중고 마켓에서는 판매자가 시세를 부풀리거나(바가지), 초보 구매자가 물건의 정확한 가치를 알지 못해 사기를 당하는 사례가 빈번했습니다. 본 프로젝트는 사용자가 아이템과 상태를 입력하면, 내장된 평가 알고리즘 엔진이 적정 시장가를 계산하고 구매 위험도를 분석하여 시각적인 리포트로 제공합니다.

## 2. 해결하고자 하는 문제 (Pain Point)
*   **시세 검색의 복잡성**: 이베이, 당근마켓, 크림(KREAM) 등 플랫폼마다 가격이 달라 평균 시세를 파악하기 어려움.
*   **상태에 따른 가격 변동 무지**: 카드의 흠집, 개봉 여부, 공인 감정 기관(PSA)의 등급에 따라 가격이 10배 이상 차이 남에도 불구하고, 초보자는 이를 알지 못함.
*   **악의적인 시세 조작**: 특정 유저들이 담합하여 가격을 올리는 덤터기(바가지) 판매 성행.

## 3. 핵심 기능 (Core Features)
*   **직관적인 아이템 정보 입력 (Input)**: 불필요한 타이핑 없이, 미리 구축된 데이터베이스 내에서 카테고리와 품목, 상태, 등급을 드롭다운으로 빠르게 선택.
*   **실시간 AI 가치 평가 (Real-time Evaluation)**: 외부 서버 통신 지연 없이 브라우저 내장 자바스크립트 엔진(`app.js`)을 활용해 즉각적인 적정가(Estimated Price) 도출.
*   **구매 위험도 판별 뱃지 (Safety Badge)**: 도출된 적정가와 비교하여 현재 거래가가 매력적인지, 혹은 사기(바가지)인지 4단계(🟢 저렴, 🔵 적정, 🟡 주의, 🔴 바가지) 뱃지로 시각화하여 경고.
*   **종합 분석 리포트 자동 생성**: 시세 변동성(Volatility) 지표를 바탕으로 "현재는 고점입니다. 관망을 추천합니다" 등 텍스트 기반의 투자 가이드 제공.

## 4. 기술 스택 (Technology Stack)
본 프로젝트는 **Zero-dependency, Stand-alone 프론트엔드 환경**을 목표로 개발되었습니다. 서버 구축 및 API 비용 없이 어디서든 즉시 실행 가능합니다.

*   **Frontend**: HTML5, Vanilla CSS3 (Custom Responsive Design), Vanilla JavaScript (ES6+)
*   **Design**: 깔끔한 흰색 배경에 파랑색, 노란색, 초록색을 포인트 컬러로 활용한 모던 UI/UX
*   **Data Processing**: Client-side JSON Structure Modeling
*   **Deployment**: Local Web Server (Python `http.server`) & GitHub Repository

## 5. 비즈니스 모델 및 향후 발전 가능성 (Next Step)
본 프로젝트는 대학 과제 및 프로토타입(MVP) 수준으로 구현되었으나, 추후 상용화 시 다음과 같은 방향으로 수익 모델을 창출할 수 있습니다.
*   **수집품 전용 에스크로 결제 수수료**: 적정가 판별 후 해당 플랫폼 내에서 직접 안전거래를 진행하고 중개 수수료(약 2~3%) 취득.
*   **전문 감정 대행 서비스 연결**: '미감정' 물품을 보유한 유저를 전문 감정 센터(BGS, PSA 등)와 연결해주고 제휴 수수료 수취.
*   **프리미엄 빅데이터 API 리셀**: 축적된 시세 데이터를 바탕으로 타 중고거래 플랫폼(당근마켓 등)에 시세 판별 API를 B2B로 납품.

## 실행 순서 (터미널에 한 줄씩 복사 → 붙여넣기 → 엔터)

요구 사항: 컴퓨터에 **Python 3.x** 이상이 설치되어 있어야 합니다.

1. **파일이 있는 폴더로 이동**
   ```bash
   cd ~/Desktop/FairPrice-AI
   ```
   *(Windows 운영체제인 경우: `cd Desktop\FairPrice-AI`)*

2. **로컬 웹 서버 실행**
   ```bash
   python -m http.server 8000
   ```
   *(Mac인 경우 `python3 -m http.server 8000` 입력)*

3. **웹 브라우저 접속**
   터미널에 `Serving HTTP on :: port 8000 ...` 이라는 메시지가 떴다면 **크롬 브라우저** 주소창에 아래 주소를 복사해서 들어가세요.
   > **http://localhost:8000**

## 자주 나는 오류 (Troubleshooting)

- **`"python: command not found"`** 👉 [https://www.python.org](https://www.python.org) 에서 Python을 설치하세요. (Windows는 설치 시 반드시 'Add python.exe to PATH' 체크박스를 체크해야 합니다.)
- **`"Address already in use"`** 👉 이미 8000번 포트가 사용 중입니다. 명령어 맨 끝 숫자를 바꿔서 `python -m http.server 8080` 으로 실행 후 `http://localhost:8080` 에 접속하세요.
- **화면 디자인이 깨지거나 CSS가 적용 안 될 때** 👉 브라우저 캐시 문제일 확률이 높습니다. `Ctrl + Shift + R` (맥은 `Cmd + Shift + R`)을 눌러 강력 새로고침을 진행하세요.

## AI 기능 명세 (Core Logic)

심화 가격 분석 및 투자의견 작성은 `app.js` 내부에 구현된 **자체 추론 엔진**을 통해 브라우저에서 직접 연산됩니다. 외부 API(OpenAI 등) 연결 없이도 동작하는 완전 독립형(Stand-alone) 구조입니다.

*   **상태 가중치 매핑**: 사용자가 선택한 아이템의 희소성, 개봉 여부(Mint, Played), 공식 등급(PSA/BGS) 데이터를 기반으로 베이스 가격에 곱연산(Multiplier) 적용.
*   **변동성(Volatility) 기반 모멘텀 평가**: 가상 알고리즘(Randomized Hash Modeling)을 통해 품목별로 6개월간의 가상 시세 등락폭을 생성하고, 이를 기반으로 **[저렴/바가지/적정/주의]** 뱃지를 자동 생성 및 투자 가이드 제공.
